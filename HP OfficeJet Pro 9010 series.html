Embedded Web Server (EWS) Language

To view the EWS in another language, select the desired language from the list.

Note:
The language you select will be used when displaying the EWS in this web browser, on this computer. However, when the EWS is opened on other computers, it will be displayed in the default language for the web browser being used. To enable the EWS to be displayed in a specific language in all web browsers, on all computers, visit the Embedded webserver (EWS) language settings page.
